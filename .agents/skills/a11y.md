# Skill: Acessibilidade (A11Y)

> **Canônico:** [`docs/A11Y.md`](../../docs/A11Y.md) · Tokens: [`design/tokens.md`](../../design/tokens.md) · Entrada: [`docs/CANON.md`](../../docs/CANON.md)

## Descrição

Regras de acessibilidade para a landing. Meta: Lighthouse a11y **100** (baseline recente ~96 — revalidar).

## Contraste (tokens atuais)

| Uso | Token | Hex | Fundo | Mínimo |
|-----|-------|-----|-------|--------|
| Texto forte | `--ink` | `#07111f` | Editorial `#f5f2ed` | 4.5:1 ✅ |
| Texto secundário | `--muted-foreground` | `#556070` | Editorial / Card | 4.5:1 ✅ |
| Eyebrows / labels dourados | `--warm-accent-contrast` | `#7a5a1a` | Card / Editorial | 4.5:1 ✅ |
| Acento decorativo | `--warm-accent` | `#c9983f` | **Não** usar como texto pequeno em fundo claro | — |
| Accent em dark | `--accent` / on-dark | `#16a99c` / `#5eead4` | Surface dark | OK |

## Obrigações rápidas

- Skip link `#conteudo` · nav semântica · foco `.focus-ring`
- Touch targets ≥ 44px · ESC fecha modal · `DialogTitle` / `DialogDescription`
- CTAs de demo: `aria-label` específico (`Abrir demonstração: {título}`)
- Filtros: `aria-live="polite"` · iframe com `title`
- `prefers-reduced-motion` via `MotionProvider`
- Thumbnails: `alt` descritivo (`thumbnailAlt` em `content.ts`)

## Não fazer

- Texto dourado `#c9983f` / `#d4a853` em labels pequenos sobre fundo claro
- Hex fora de `globals.css`
- Referenciar cockpit / `ProfileCockpit` (arquivado)
