# Skill: Acessibilidade

> Canônico: [`docs/A11Y.md`](../../docs/A11Y.md). Baseline local: Lighthouse a11y 100 desktop/mobile.

- Texto 4,5:1; componentes e texto grande 3:1.
- Texto teal/dourado em claro usa variantes `Contrast`.
- Nome acessível inclui o rótulo visível; não o substitua por descrição diferente.
- Skip link, foco `.focus-ring`, ESC e retorno ao acionador são obrigatórios.
- Touch target ≥44 px; texto mobile ≥14 px.
- Ícone com texto usa `aria-hidden`; ícone sozinho recebe nome.
- Filtros anunciam contagem via `aria-live`.
- Âncora não usa iframe; complementar usa iframe com `title`.
- ECharts recebe resumo textual; MapLibre usa região nomeada e atribuição.
- Motion respeita `prefers-reduced-motion`; hero LCP é estático.

Antes de concluir: teclado, 200% zoom, 375 px, modal, filtros, `npm run test:e2e` e `npm run lighthouse:all`.
